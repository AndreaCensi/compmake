General configuration
interactive True Whether we are in interactive mode (e.g., ask confirmations)
Appearance
colorize True Use colors in terminals if possible.
Job execution
echo_stdout True If true, the job output to stdout is shown.
echo_stderr True If true, the job output to stderr is shown.
save_progress True Whether to save intermediate results for jobs that use the yield() paradigm. Automatically disabled for cluster slaves to save bandwidth.
