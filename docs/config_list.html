General configuration
check_params False If true, erases the cache if job parameters appear to change. Very useful but you need to define __eq__() in all the objects you use as parameters.
db filesystem Specifies db backend. Options: 'filesystem', 'redis'. XXX: so far, only honored at startup time.
interactive True Whether we are in interactive mode (e.g., ask confirmations).
Job execution
echo_stdout True If true, the job output to stdout is shown.
echo_stderr True If true, the job output to stderr is shown.
save_progress True Whether to save intermediate results for jobs that use the yield() paradigm. Automatically disabled for cluster slaves to save bandwidth.
Cluster execution
cluster_conf cluster.yaml Location of cluster configuration file.
hostname localhost Nickname for current host (set by compmake master).
cluster_nice 0 Nice level for spawned remote processes.
cluster_show_cmd True If true, it shows the connection string to the slaves.
Redis backend
redis_host localhost Hostname[:port] for Redis host
Filesystem backend
path compmake_storage [filesystem db] Path to directory for filesystem storage.
Appearance
colorize True Use colors in terminals if possible.
