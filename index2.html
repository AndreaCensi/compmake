Compmake¶

compmake is make for batch python processes. It is a non-obtrusive module that provides:

  • make–like facilities to your computations (make, clean, etc.), including caching of temporary results. That is you can do a CTRL+C, play your videogame, and then restart compmake without losing data.
  • Single-host parallelization using the multiprocessing module.
  • Multiple-host parallelization using ssh-spawned slaves.

Quick installation¶

The quick install is:

$ easy_install compmake

This will allow you to run compmake on a single host. However, there are also separate dependencies to install for some advanced features. See install.rst__ for more information.

Source download¶

You can download this project in either zip or tar formats.

You can also clone the project with Git by running: 

$ git clone git://github.com/AndreaCensi/compmake
