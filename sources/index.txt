compmake
========

``compmake`` is ``make`` for batch python processes.

.. image:: workers.jpg
   :align: right

It is a non-obtrusive module that allows to add:

* ``make``--like facilities to your computations (``make``, ``clean``, etc.)

* Automatic **caching** of the results on disk.

* **Parallelization** using the ``multiprocessing`` module.

Installation
------------

The quick install is::

$ easy_install compmake

Altough there are also dependencies to install for advanced features. 
See install.rst__ for more information.
