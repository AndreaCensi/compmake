Parallel equivalent of make.

Uses multiprocessing.Process as a backend and a Python queue to communicate with the workers.

Options:

parmake n=10 Uses 10 workers parmake recurse=1 Recursive make: put generated jobs in the queue. parmake new_process=1 Run the jobs in a new Python process. parmake echo=1 Shows the output of the jobs. This might slow down everything.

parmake new_process=1 echo=1 Not supported yet.
