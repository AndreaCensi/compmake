graph Module¶

compmake.plugins.graph.graph(job_list, filename='compmake', compact=0, filter='dot', format='png')¶

Creates a graph of the given targets and dependencies

graph filename=filename compact=0,1 format=png,...

Params:

filename: name of generated filename in the dot format compact=0: whether to include the job names in the nodes filter=[dot,circo,twopi,...] which algorithm to use to arrange

the nodes. This depends on the topology of your computation. The default is ‘dot’ (hierarchy top-bottom).

format=[png,...] The output file format.
