compmake.events.registrar.remove_all_handlers()¶

Removes all event handlers. Useful when events must not be processed locally but routed to the original process somewhere else.

compmake.events.registrar.register_fallback_handler(handler)¶

Registers an handler who is going to be called when no other handler can deal with an event. Useful to see if we are ignoring some event.

compmake.events.registrar.register_handler(event_name, handler)¶

Registers an handler with an event name. The event name might contain asterisks. “*” matches all.
