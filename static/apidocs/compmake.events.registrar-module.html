register_fallback_handler(handler)

source code 
Registers an handler who is going to be called when no other handler can deal with an event. Useful to see if we are ignoring some event.

register_handler(event_name, handler)

source code 
Registers an handler with an event name. The event name might contain asterisks. "*" matches all.

remove_all_handlers()

source code 
Removes all event handlers. Useful when events must not be processed locally but routed to the original process.
