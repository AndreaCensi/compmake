comp(command, *args, **kwargs)

source code 

Main method to define a computation step.

Extra arguments:

:arg:job_id: sets the job id (respects job_prefix) :arg:extra_dep: extra dependencies (not passed as arguments)

Raises UserError if command is not pickable.

interpret_commands(commands_str, separator=';')

source code 

Interprets what could possibly be a list of commands (separated by ";") If one command fails, it returns its retcode, and then the rest are skipped.

Returns 0 on success; else returns either an int or a string describing what went wrong.
